## 6. TECHNICAL NOTES

### Construction Sequence

1. Layout column locations per plan (verify 1'-0" end clearance)
2. Core drill for epoxy anchors (verify slab condition adequate)
3. Install base plates with Hilti HY200 anchors
4. Erect HSS columns and verify plumb
5. Frame steel stud wall and fasten to HSS columns both sides
6. Install sheathing/backing for finish
7. Apply PL3 Danish Maple finish
8. Install stainless steel cap
9. Install rubber cove base
10. Install corner guards

## 7. SUMMARY OF SCOPE

This PCN adds two 54" high free-standing pony walls at the Customer Service Desk area (Room 103). The walls will be constructed with 6" steel studs supported by HSS 76x76x6.4 columns at maximum 10'-0" spacing, anchored to the concrete slab with epoxy anchors. Walls will receive PL3 Danish Maple finish to match existing Customer Service area, stainless steel cap at top, rubber cove base at bottom, and corner guards as required.
